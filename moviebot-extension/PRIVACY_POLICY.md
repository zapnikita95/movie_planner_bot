# Политика конфиденциальности расширения Movie Planner Bot

## Сбор данных пользователей

**Мы НЕ собираем пользовательские данные для своих целей.**

Все данные хранятся локально на устройстве пользователя:
- `linked_chat_id` - ID чата/группы пользователя в Telegram (хранится в `chrome.storage.local`)
- `linked_user_id` - ID пользователя в Telegram (хранится в `chrome.storage.local`)
- `has_tickets_access` - статус подписки на билеты (хранится в `chrome.storage.local`)

Эти данные используются исключительно для связи расширения с вашим личным ботом в Telegram.

## Внешние запросы

Расширение отправляет запросы только на сервер пользователя (Railway), указанный в коде:
- Проверка кода привязки
- Получение информации о фильмах
- Добавление фильмов в базу
- Планирование просмотров
- Проверка подписки

Все данные отправляются на сервер, принадлежащий пользователю/владельцу бота, и не используются третьими лицами.

## Удаленный код

Расширение **НЕ использует** удаленный код (remote code). Весь JavaScript код находится в расширении и выполняется локально.

## Права доступа

- **storage** - для хранения локальных настроек (chat_id, user_id, статус подписки)
- **activeTab** - для определения текущей страницы браузера
- **contextMenus** - для пункта меню "Отправить в Movie Planner Bot"
- **notifications** - для уведомлений о действиях с билетами
- **tabs** - для получения информации о текущей вкладке и отправки сообщений content scripts
- **host_permissions** - для доступа к сайтам с информацией о фильмах и к билетным сайтам (для извлечения данных и добавления кнопок)
